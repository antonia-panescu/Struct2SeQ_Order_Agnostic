# Eterna100-V2: solved across folding oracles

Same 4 models, identical generated sequences (K=1000 budget); only the fold-back oracle differs. ViennaRNA 2 is the reference engine the V2 targets were tuned for; all 100 V2 targets are pseudoknot-free.
